# Style System Stories

This folder is reserved for Storybook runtime validation of non-componentized visual rules, following `docs/storybook-classification.md`.

## Belongs Here

- Theme palette validation
- Typography scale validation
- Spacing and density checks
- Radius policy checks
- Data color and chart rule validation
- Semantic token previews after evidence supports the token names

## Does Not Belong Here

- React component implementations
- Product page compositions
- Real Figma node claims without source evidence
- Final raw token values before evidence and review
- Experiments promoted into stable rules without a decision record

No story files are created in this preparation step.
