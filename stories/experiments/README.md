# Experiment Stories

This folder is reserved for unstable migration and de-branding experiments, following `docs/storybook-classification.md`.

## Belongs Here

- ReducedAppleStyle validation
- WebAdaptation validation
- ChineseTypography validation
- NoRadiusPolicy validation
- FocusAppMigration and LearningTrackerMigration experiments
- Clearly labeled migration tests that have not become stable design-system rules

## Does Not Belong Here

- Stable component or pattern stories
- Final token decisions
- Experiments silently promoted into the style system
- Claims of source evidence without mapped Figma nodes or screenshots
- UI implementation beyond the explicit experiment boundary

No story files are created in this preparation step.
